## About The Project

### this is the backend code for the website of gananaad , it connects the database to the main website as well as admin dashboard.

<p align="right">(<a href="#readme-top">back to top</a>)</p>



### Built With

* NodeJs
* express
* mongodb
* mongoose
* jwt
